## 🚨 PROBLEMAS MÁS FRECUENTES

### ❌ "No se abre el manual al hacer doble-click"

**💡 SOLUCIONES:**
1. **Click derecho** en el archivo .bat → "Ejecutar como administrador"
2. **Desactiva temporalmente** el antivirus y vuelve a intentar
3. **Verifica** que el archivo no esté bloqueado (Click derecho → Propiedades → Desbloquear)
4. **Usa la OPCIÓN 1 PORTABLE** que es más compatible

**🔧 CAUSA:** Restricciones de seguridad del sistema o antivirus

---

### ❌ "Python no encontrado" o "Node.js no encontrado"

**💡 SOLUCIONES INMEDIATAS:**
1. **USA OPCIÓN 1 PORTABLE** - No necesita Python ni Node.js
   - Ve a `OPCION_1_PORTABLE`
   - Doble-click en `ABRIR_MANUAL_HPV3.bat`

**💡 PARA INSTALAR PYTHON (Si quieres usar Opción 2):**
1. **Microsoft Store** (MÁS FÁCIL):
   - Abre Microsoft Store
   - Busca "Python 3.11"
   - Instala y reinicia el .bat

2. **Sitio oficial:**
   - Ve a https://www.python.org/downloads/
   - Descarga Python 3.x
   - **IMPORTANTE:** Marca "Add Python to PATH" durante instalación

**🔧 CAUSA:** Software no instalado en el sistema

---

### ❌ "Se ve mal en el navegador" o "Diseño roto"

**💡 SOLUCIONES:**
1. **Presiona F11** para modo pantalla completa
2. **Presiona Ctrl + 0** para resetear el zoom al 100%
3. **Actualiza la página** con F5
4. **Cambia de navegador**: Chrome → Edge → Firefox
5. **Verifica resolución de pantalla** (mínimo 1024x768)

**🔧 CAUSA:** Zoom incorrecto o navegador incompatible

---

### ❌ "No funciona el quiz" o "No guarda las respuestas"

**💡 SOLUCIONES:**
1. **Actualiza la página** (F5) y vuelve a la sección Evaluación
2. **Limpia caché** del navegador:
   - Chrome: Ctrl+Shift+Supr → Eliminar todo
   - Edge: Ctrl+Shift+Supr → Eliminar todo
3. **Usa modo incógnito/privado**
4. **Cambia de navegador**

**🔧 CAUSA:** Caché corrupto o JavaScript bloqueado

---

### ❌ "Perdí mi progreso"

**💡 EXPLICACIÓN:**
- El progreso se guarda **automáticamente durante la sesión**
- Se pierde al **cerrar el navegador** (es normal)
- No es un error, es el comportamiento diseñado

**💡 SOLUCIONES:**
1. **Completa el manual en una sesión** (75-105 minutos)
2. **Exporta tu progreso** desde el menú de ayuda antes de cerrar
3. **Deja el navegador abierto** si necesitas pausas largas

---

### ❌ "Puerto 8000 ocupado" (Solo Opción 2)

**💡 SOLUCIONES:**
1. **Ve directamente** a http://localhost:8000 en tu navegador
2. **Cierra otros programas** que usen servidores web locales
3. **Reinicia el equipo** si persiste el problema
4. **Usa OPCIÓN 1 PORTABLE** como alternativa

**🔧 CAUSA:** Otro programa usando el mismo puerto

---

### ❌ "El archivo .bat se cierra inmediatamente"

**💡 SOLUCIONES:**
1. **Abre CMD como administrador**
2. **Arrastra el archivo .bat** a la ventana CMD
3. **Presiona Enter** para ver el error específico
4. **Ejecuta desde el Explorador de archivos** como administrador

**🔧 CAUSA:** Error en la ejecución que no se muestra
