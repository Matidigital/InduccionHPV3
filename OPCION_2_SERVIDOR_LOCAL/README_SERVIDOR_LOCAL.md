### 🔧 ¿PROBLEMAS? SOLUCIONES EXPERTAS

**Problema:** "Python/Node.js no encontrado"
- **Solución Rápida:** Usa `OPCION_1_PORTABLE\ABRIR_MANUAL_HPV3.bat`
- **Solución Completa:** Instala Python desde Microsoft Store o Node.js desde nodejs.org

**Problema:** "Puerto 8000 ocupado"
- **Solución:** Cierra otros servidores web locales, o ve a http://localhost:8000 directamente

**Problema:** "No se abre el navegador"
- **Solución:** Ve manualmente a http://localhost:8000 en tu navegador

**Problema:** "Error de permisos"
- **Solución:** Ejecuta como administrador (click derecho → "Ejecutar como administrador")
